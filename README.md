# Análise de Churn em Telecomunicações (SaaS)

## Sobre o Projeto
Este projeto tem como objetivo realizar a análise de retenção de clientes (Churn Analysis) para uma empresa de telecomunicações. Através de técnicas de Business Intelligence, transformamos dados brutos em insights estratégicos para apoiar a tomada de decisão.

## Tecnologias e Ferramentas
* **Ferramenta:** Power BI Desktop
* **Modelagem:** Data Modeling (Star Schema)
* **Linguagem:** DAX (Data Analysis Expressions) & Power Query (M)

## Estrutura do Repositório
* `/dashboard`: Contém o arquivo final `.pbix` do projeto.
* `/data`: Contém o dataset utilizado (`telecom_customer_churn.csv`).
* `/docs`: Documentação complementar.

## Metodologia de Trabalho
1. **Coleta de Dados:** Dataset obtido via Kaggle.
2. **Tratamento (ETL):** Limpeza e normalização no Power Query para garantir a integridade.
3. **Modelagem:** Estruturação em *Star Schema* para otimização de performance e cálculos.
4. **Visualização:** Dashboards focados em métricas de risco e comportamento do cliente.

## Status do Projeto
O projeto está em fase de modelagem e desenvolvimento de medidas DAX para cálculo de Churn Rate e análise de coortes.
