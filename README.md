# Dashboard de Business Intelligence para Análise de Retenção de Clientes (Churn Rate) 📉

Este projeto apresenta o desenvolvimento de uma solução de Business Intelligence focada no monitoramento, análise e diagnóstico de indicadores de cancelamento de clientes (Churn). O objetivo é apoiar decisões estratégicas de retenção de receita em uma empresa de serviços.

## 🛠️ Tecnologias e Conceitos Aplicados
- **Power BI / Excel** (Criação de dashboards dinâmicos)
- **Power Query** (Extração, transformação e higienização dos dados)
- **Modelagem de Dados** (Estruturação em Star Schema com tabelas Fato e Dimensão)
- **Métricas Gerenciais** (Uso de fórmulas/DAX para cálculo de Churn Rate e perdas financeiras)

## 🏗️ Estrutura da Solução
- Tratamento e cruzamento de dados de contratos, pagamentos e perfis de clientes.
- Construção de visuais interativos para rápida identificação de clientes com alto risco de evasão.

## 📈 Resultados Obtidos
Disponibilização de uma ferramenta visual e interativa para suporte à tomada de decisão estratégica e retenção de receita.
